3.3 技術ポートフォリオの構築

📁 ポートフォリオ用語の徹底解説

  • GitHub: プログラムのソースコードを保存・管理するオンラインサービス

  • README: プロジェクトの説明書。何を作ったか、どう使うかを記載
  • コミット: 変更内容を記録に残すこと（セーブのようなもの）
  • 技術スタック: プロジェクトで使用した技術の一覧

GitHub活用法

function showAnswer(questionId) {
  const answer = document.getElementById('answer' + questionId);
  if (answer.style.display === 'none' || answer.style.display === '') {
    answer.style.display = 'block';
  } else {
    answer.style.display = 'none';
  }
}

function showExerciseAnswer(exerciseId) {
  const answer = document.getElementById('exerciseAnswer' + exerciseId);
  if (answer.style.display === 'none' || answer.style.display === '') {
    answer.style.display = 'block';
  } else {
    answer.style.display = 'none';
  }
}
