2.1 総合演習の意義と実践アプローチ（60分）
実業務での複合的スキル要求

現代の業務環境では単一技術ではなく、複数のプロンプト技術を組み合わせた統合的アプローチが求められます。

  • • Chain of Thought + データ分析プロンプトの組み合わせ
  • • 要約技術 + コード生成の実践的統合
  • • リスク評価 + 意思決定支援の複合活用
システム思考によるプロンプト設計

個々の問題を孤立して解決するのではなく、業務プロセス全体を俯瞰した設計アプローチを学習します。

入力 → 処理 → 出力 → フィードバック → 改善

function toggleAnswer(questionId) {
    const element = document.getElementById(questionId + '-answer');
    const button = event.target;
    
    if (element.classList.contains('hidden')) {
        element.classList.remove('hidden');
        button.textContent = '回答を非表示';
        button.classList.add('bg-red-500', 'hover:bg-red-600');
        button.classList.remove('bg-blue-500', 'hover:bg-blue-600');
    } else {
        element.classList.add('hidden');
        button.textContent = '回答を表示';
        button.classList.add('bg-blue-500', 'hover:bg-blue-600');
        button.classList.remove('bg-red-500', 'hover:bg-red-600');
    }
}
