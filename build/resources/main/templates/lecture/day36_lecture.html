問題1: Java基礎総復習

以下のJavaコードについて、実行結果を予測してください。

public class Test {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("JavaScript");
        list.stream()
            .filter(s -> s.startsWith("Java"))
            .forEach(System.out::println);
    }
}

実行結果:

Java
JavaScript

解説: Stream APIのfilter()メソッドで"Java"で始まる文字列のみを抽出し、forEach()で出力します。

問題2: データベース正規化

問題3: アルゴリズム解析

バブルソートの最悪時計算量は何ですか？理由とともに答えてください。

答え: O(n²)

理由: 最悪の場合（逆順でソートされたデータ）、外側のループがn回、内側のループが平均してn/2回実行されるため、全体でn×n/2 ≈ n²/2 = O(n²)となります。

問題4: セキュリティ対策

SQLインジェクション攻撃を防ぐための対策を3つ挙げてください。

対策:

  1. 1. プリペアードステートメントの使用: パラメータ化クエリでSQL文と データを分離
  2. 2. 入力値検証: ユーザー入力の形式・文字種・長さをチェック
  3. 3. エスケープ処理: 特殊文字（', ", \など）を無害化

問題5: プロジェクト管理

プロジェクトの進捗管理でよく使用される「ガントチャート」の特徴と利点を説明してください。

ガントチャートの特徴:

  • • 横軸に時間、縦軸にタスクを配置した棒グラフ
  • • タスクの開始・終了日程が視覚的に分かる
  • • タスク間の依存関係を表現可能

利点:

  • • プロジェクト全体の進捗が一目で把握できる
  • • 遅延タスクの影響範囲が分かる
  • • リソース配分の最適化が可能

【午前試験】問題1: コンピュータの基本構成

CPUの主な機能として最も適切なものはどれか。

正解: イ

CPU（Central Processing Unit）の主な機能は、プログラムの命令を実行し、演算処理を行うことです。制御装置と演算装置から構成されます。

【午前試験】問題2: データベース

関係データベースにおいて、主キーの制約として適切でないものはどれか。

正解: ウ

外部キー制約は主キーの制約ではありません。主キーには一意性制約、NOT NULL制約、エンティティ整合性制約が適用されます。

【午後試験】問題1: アルゴリズム（情報セキュリティ）

以下の暗号化アルゴリズムについて、公開鍵暗号方式はどれか。

正解: ウ

RSAは公開鍵暗号方式です。AES、DES、RC4は共通鍵暗号方式（対称鍵暗号方式）です。公開鍵暗号では暗号化と復号化に異なる鍵を使用します。

【午後試験】問題2: プログラミング（Java）

以下のJavaコードの実行結果を選択してください。

public class Test {
    public static void main(String[] args) {
        int[] array = {3, 1, 4, 1, 5};
        Arrays.sort(array);
        System.out.println(Arrays.toString(array));
    }
}

正解: イ

Arrays.sort()メソッドは配列を昇順でソートします。[3, 1, 4, 1, 5]が[1, 1, 3, 4, 5]にソートされ、Arrays.toString()で文字列表現として出力されます。

【総合問題】プロジェクト管理

プロジェクトのリスク管理において、「リスクの影響度」と「発生確率」を組み合わせた評価マトリックスを何と呼ぶか。

正解: イ

リスクマトリックスは、リスクの発生確率と影響度を組み合わせてリスクの重要度を評価するツールです。通常、縦軸に影響度、横軸に発生確率を配置し、リスクレベルを色分けで表現します。
