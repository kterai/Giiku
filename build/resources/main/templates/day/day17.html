15:30 - 16:45

疑問点の解消と明日の最終模擬試験への準備を行います。

今日のポイント

改善の測定

2回目の模擬試験では、前回の経験を活かして時間配分を改善し、より確実に解答できるようにしましょう。特に前回間違えた分野での改善を重点的に確認します。

注意すべき問題

「誤っているものを選べ」などの否定形の問題や、複数の正解がある問題には特に注意が必要です。問題文を最後まで丁寧に読む習慣を身につけましょう。

Java17新機能

試験直前の対策として、Java SE 17の新機能に関する問題や、経験上多くの受験者が間違えやすいトピックに焦点を当てて学習します。

重点対策分野

Java Silver頻出問題パターン

基本文法・制御構文

  • switch文の新構文（Java 17対応）
  • for-each文の制約と例外ケース
  • 演算子の優先順位と評価順序
  • 変数のスコープと初期化タイミング

オブジェクト指向

  • 継承とポリモーフィズムの動作
  • 抽象クラスとインターフェースの使い分け
  • static修飾子の影響範囲
  • コンストラクタの呼び出し順序

例外処理

  • checked例外とunchecked例外の区別
  • try-with-resources文の正しい使用法
  • finallyブロックの実行タイミング
  • 例外の伝播と処理の委譲

コレクション・Stream API

  • List、Set、Mapの使い分けと特性
  • ラムダ式の構文と型推論
  • Stream操作の中間操作と終端操作
  • Optional型の適切な使用方法
