テーマ詳細

Java 17の主な新機能

レコードクラス (Java 16)

レコードは、データを単純に保持するためのクラスを簡潔に定義するための機能です。equals(), hashCode(), toString()メソッドや、すべてのフィールドに対するコンストラクタとgetterが自動生成されます。

// 従来の方法（ボイラープレートが多い） public class Point { private final int x; private final int y; public Point(int x, int y) { this.x = x; this.y = y; } public int getX() { return x; } public int getY() { return y; } @Override public boolean equals(Object o) { if (this == o) return true; if (o == null || getClass() != o.getClass()) return false; Point point = (Point) o; return x == point.x && y == point.y; } @Override public int hashCode() { return Objects.hash(x, y); } @Override public String toString() { return "Point[x=" + x + ", y=" + y + "]"; } } // レコードを使った方法（簡潔） public record Point(int x, int y) { // コンパクト！すべてが自動生成される // コンパクトコンストラクタ（バリデーションなどを追加可能） public Point { if (x < 0 || y < 0) { throw new IllegalArgumentException("座標は正の値である必要があります"); } } // 追加メソッドも定義可能 public double distance() { return Math.sqrt(x * x + y * y); } }

シールドクラス (Java 17)

シールドクラスは、どのクラスがそのクラスを継承できるかを制限する機能です。クラス階層を制御し、パターンマッチングと組み合わせることで強力な型安全性を提供します。

// シールドクラスの定義 public sealed class Shape permits Circle, Rectangle, Triangle { // 共通のフィールドとメソッド private String color; public Shape(String color) { this.color = color; } public String getColor() { return color; } public abstract double area(); } // 許可されたサブクラスの実装 public final class Circle extends Shape { private double radius; public Circle(String color, double radius) { super(color); this.radius = radius; } @Override public double area() { return Math.PI * radius * radius; } } public final class Rectangle extends Shape { private double width; private double height; public Rectangle(String color, double width, double height) { super(color); this.width = width; this.height = height; } @Override public double area() { return width * height; } } public non-sealed class Triangle extends Shape { // non-sealedにすると、このクラスを継承できるようになる private double base; private double height; public Triangle(String color, double base, double height) { super(color); this.base = base; this.height = height; } @Override public double area() { return 0.5 * base * height; } }

パターンマッチング for instanceof (Java 16)

instanceofと型キャストを1つのステップで行える構文で、コードをより簡潔で安全にします。

// 従来の方法 if (obj instanceof String) { String s = (String) obj; // キャストが必要 System.out.println(s.length()); } // パターンマッチング for instanceof if (obj instanceof String s) { // 型チェックとキャストを同時に行う System.out.println(s.length()); // sはString型として使用可能 } // シールドクラスとパターンマッチングの組み合わせ public double calculateArea(Shape shape) { if (shape instanceof Circle c) { return Math.PI * c.getRadius() * c.getRadius(); } else if (shape instanceof Rectangle r) { return r.getWidth() * r.getHeight(); } else if (shape instanceof Triangle t) { return 0.5 * t.getBase() * t.getHeight(); } else { throw new IllegalArgumentException("未知の図形タイプです"); } }

テキストブロック (Java 15)

複数行の文字列リテラルを、インデントや改行を保持したまま簡潔に表現できる機能です。

// 従来の方法（読みにくい） String html = "<html>\n" + " <body>\n" + " <p>Hello, World!</p>\n" + " </body>\n" + "</html>\n"; // テキストブロック（読みやすい） String html = """ <html> <body> <p>Hello, World!</p> </body> </html> """; // SQLクエリの例 String query = """ SELECT id, name, email FROM users WHERE status = 'active' ORDER BY name LIMIT 10 """;

スイッチ式 (Java 14)

スイッチを式として使用でき、値を返すことが可能になりました。また、アロー演算子（->）を使った簡潔な構文も導入されました。

// 従来のスイッチ文 String dayType; switch (day) { case "MONDAY": case "TUESDAY": case "WEDNESDAY": case "THURSDAY": case "FRIDAY": dayType = "Weekday"; break; case "SATURDAY": case "SUNDAY": dayType = "Weekend"; break; default: dayType = "Invalid day"; } // 新しいスイッチ式 String dayType = switch (day) { case "MONDAY", "TUESDAY", "WEDNESDAY", "THURSDAY", "FRIDAY" -> "Weekday"; case "SATURDAY", "SUNDAY" -> "Weekend"; default -> "Invalid day"; }; // yield を使った複数行の処理 int numLetters = switch (day) { case "MONDAY", "FRIDAY", "SUNDAY" -> { System.out.println(day + "の処理"); yield 6; } case "TUESDAY" -> { yield 7; } case "THURSDAY", "SATURDAY" -> { yield 8; } case "WEDNESDAY" -> { yield 9; } default -> { throw new IllegalArgumentException("不正な曜日: " + day); } };
