詳細スケジュール

10:00-10:30 フルスタック開発概要・プロジェクト構成

フルスタック開発の全体像理解。フロントエンド・バックエンド分離アーキテクチャの利点、API ファーストアプローチの重要性、マイクロサービス基礎概念を学習。プロジェクト構成パターン（モノレポ vs マルチレポ）の比較検討と最適な選択基準を習得する。

10:30-11:30 React × Spring Boot 連携基礎

開発環境構築とプロキシ設定。Create React App と Spring Boot の統合方法、Hot Reload 設定、CORS設定の詳細を実践。Axios を活用したAPIクライアント実装、エラーハンドリング、リクエスト・レスポンス変換の実装パターンを習得する。

11:30-12:30 JWT認証システム統合実装

Spring Security による JWT 認証システム構築とReactでの認証状態管理。アクセストークン・リフレッシュトークンの実装、プロテクトルートの設定、自動リダイレクト機能、認証エラーハンドリングの包括的な実装方法を習得する。

12:30-13:30 昼休憩
13:30-14:30 状態管理・フォーム統合・ファイル処理

Redux Toolkit による状態管理、React Hook Form によるフォーム処理、サーバーサイドバリデーション連携。ファイルアップロード機能（multipart/form-data、プログレスバー、ドラッグ&ドロップ、画像プレビュー）の実装技術を総合的に習得する。

14:30-15:00 リアルタイム通信・パフォーマンス最適化

WebSocket と Server-Sent Events によるリアルタイム通信実装、通知システム構築。コード分割、遅延読み込み、キャッシュ戦略によるパフォーマンス最適化技術、バンドルサイズ削減手法を実践的に学習する。

テーマ詳細

フルスタック開発アーキテクチャの完全理解

本日は現代的なWebアプリケーション開発の核心となるフルスタック開発技術を習得します。 React によるモダンなフロントエンド開発と Spring Boot による堅牢なバックエンド開発を 効率的に連携させる技術を、実践的なプロジェクトを通じて総合的に学習します。

重要なアーキテクチャ原則

  • • 関心の分離: フロントエンドとバックエンドの責任を明確に分離
  • • API ファースト: APIを中心とした設計アプローチの採用
  • • スケーラビリティ: 将来の拡張性を考慮した構造設計
  • • セキュリティ: 多層防御によるセキュリティ対策の実装

技術スタック統合ポイント

フロントエンド (React)
  • • TypeScript による型安全性
  • • React Hooks による状態管理
  • • Redux Toolkit によるグローバル状態
  • • React Router による SPA ルーティング
バックエンド (Spring Boot)
  • • Spring Security による認証・認可
  • • Spring Data JPA によるデータアクセス
  • • RESTful API による統一インターフェース
  • • WebSocket による双方向通信

セキュリティ統合の実践実装

JWT (JSON Web Token) を活用した現代的な認証システムの構築を通じて、 フロントエンドとバックエンド間でのセキュアな通信を実現します。 CSRF、XSS、SQLインジェクション等の脅威に対する多層防御戦略を学習します。

セキュリティチェックポイント

  • • JWT トークンの適切な保存とライフサイクル管理
  • • HTTPS 通信の強制とセキュアなヘッダー設定
  • • Input Validation による入力値検証の実装
  • • Rate Limiting による API 攻撃対策
  • • セキュリティログの記録と監視システムの構築

使用教材・リソース

開発ツール

  • Node.js 18+ と npm/yarn
  • Java 17+ と Maven/Gradle
  • MySQL 8.0 / PostgreSQL
  • Docker Desktop
  • Postman / Thunder Client
