以下のテンプレートをコピーし、各Dayの内容に合わせて修正してGammaAIに入力してください。

以下の指示に従って、プログラミング教育用のプロフェッショナルな講義スライドを作成してください。

## スライド全体の設定
- プロフェッショナルな教育用テンプレートを使用（青と白のカラースキームが望ましい）
- 各スライドには明確な見出しをつける
- すべてのスライドに下部にスライド番号を表示
- フォントは読みやすく明確なものを選択
- 必要に応じて関連する画像やアイコン、図解を追加
- 背景色で文字が見えづらくならないように注意する

## 追加の指示
- 各スライドには説明的なノートを追加し、講師が補足情報を伝えられるようにしてください
- コードブロックには適切なシンタックスハイライトを適用してください
- 関連する画像やアイコン、図解を適切に配置してください
- コンセプトを視覚的に分かりやすく表現する図解を含めてください
- 文字は大きめ、内容は簡潔に、視認性を重視してください
- 重要なポイントは太字または色付きのテキストで強調してください

以上の指示に基づいて、プロフェッショナルな教育用スライドを作成してください。

使い方

  1. 上記のテンプレートをコピーします
  2. 各Dayの内容に合わせて以下を変更します:
    • タイトルとサブタイトル
    • 学習目標
    • トピックごとの内容と時間配分
    • コード例やサンプル
    • 演習内容
    • 理解度チェックの問題
  3. 特に重要な部分を強調したい場合は、指示文を具体的に記述します
  4. 完成したプロンプトをGammaAIに入力してスライドを生成します

Day1の例

Day1の場合は、以下のようなポイントを含めています:

  • オリエンテーション時にJDK (Amazon Corretto)とPleiades のダウンロードを開始
  • Javaの歴史と特徴の説明
  • Java実行環境の構成図（JDK, JRE, JVM）
  • テキストエディタでのHello World作成とコマンドラインでの実行
  • Pleiadesでの開発環境構築とHello World作成
