### スライド5：条件分岐 - if-else文 (10:15-11:00)
if-else文の基本構文と使用例:
```java
// if文の基本構文
if (条件) {
    // 条件が真の場合の処理
}

// if-else文
if (score >= 60) {
    System.out.println("合格");
} else {
    System.out.println("不合格");
}

// else-if文
if (score >= 80) {
    System.out.println("優");
} else if (score >= 70) {
    System.out.println("良");
} else if (score >= 60) {
    System.out.println("可");
} else {
    System.out.println("不可");
}
```
- フローチャートを使って条件分岐の流れを視覚的に説明

### スライド6：条件分岐 - switch文 (10:15-11:00 続き)
switch文の基本構文と使用例:
```java
// switch文の基本構文
switch (変数) {
    case 値1:
        // 変数が値1の場合の処理
        break;
    case 値2:
        // 変数が値2の場合の処理
        break;
    default:
        // どの値にも一致しない場合の処理
}

// 具体例
switch (day) {
    case 1:
        System.out.println("月曜日");
        break;
    case 2:
        System.out.println("火曜日");
        break;
    // ... 他の曜日 ...
    default:
        System.out.println("無効な日付");
}
```
- breakの重要性とフォールスルーの説明
- if-elseとswitch文の使い分け

### スライド9：繰り返し処理 - for文 (12:45-13:30)
for文の基本構文と使用例:
```java
// for文の基本構文
for (初期化; 条件; 更新) {
    // 繰り返し実行される処理
}

// 具体例
for (int i = 0; i < 5; i++) {
    System.out.println("カウント: " + i);
}
```
- for文の実行フローを図で説明
- for文の各部分の役割（初期化、条件、更新）

### スライド10：繰り返し処理 - while文とdo-while文 (12:45-13:30 続き)
while文とdo-while文の基本構文と使用例:
```java
// while文
int i = 0;
while (i < 5) {
    System.out.println("カウント: " + i);
    i++;
}

// do-while文
int j = 0;
do {
    System.out.println("最低1回は実行: " + j);
    j++;
} while (j < 5);
```
- while文とdo-while文の違い
- 各ループの使い分け

### スライド11：繰り返し処理 - break文とcontinue文 (12:45-13:30 続き)
break文とcontinue文の使い方:
```java
// break文の例
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // ループを終了
    }
    System.out.println(i);
}

// continue文の例
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // 次の繰り返しに進む
    }
    System.out.println(i);  // 奇数のみ表示
}
```
- break文とcontinue文の違い
- フローチャートを使った説明

### スライド13：配列の基礎 - 宣言と初期化 (14:15-15:00)
配列の宣言と初期化の方法:
```java
// 方法1: サイズを指定して宣言後、個別に要素を初期化
int[] array1 = new int[5];
array1[0] = 10;
array1[1] = 20;
// ...

// 方法2: 宣言と同時に初期化
int[] array2 = {10, 20, 30, 40, 50};

// 方法3: 宣言と初期化を別々に
int[] array3;
array3 = new int[]{10, 20, 30, 40, 50};
```
- 配列のメモリイメージを図で説明
- インデックスは0から始まることを強調

### スライド14：配列の基礎 - 操作とアクセス (14:15-15:00 続き)
配列の操作と要素へのアクセス:
```java
// 配列の要素へのアクセス
int[] scores = {85, 90, 78, 92, 88};
System.out.println("3番目の得点: " + scores[2]);  // 78

// 配列の長さ
int length = scores.length;  // 5

// for文による配列の走査
for (int i = 0; i < scores.length; i++) {
    System.out.println("得点 " + i + ": " + scores[i]);
}

// 拡張for文による配列の走査
for (int score : scores) {
    System.out.println("得点: " + score);
}
```
- 配列の長さ(length)の使い方
- 範囲外アクセスの例外(ArrayIndexOutOfBoundsException)
