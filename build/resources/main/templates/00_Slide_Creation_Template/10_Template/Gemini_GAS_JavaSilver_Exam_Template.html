Gemini用 Java Silver模擬試験GAS作成プロンプト

Google Apps Script (GAS)を使用してJava Silver模擬試験のGoogle FormとSpreadsheetを自動生成するためのプロンプト

作成日: 2025年7月29日

依頼内容

Java Silver模擬試験のGoogle FormとGoogle Spreadsheetを自動生成するためのGoogle Apps Script（GAS）コードを作成してください。このスクリプトは、実行すると指定した要件に沿ったGoogle Formを作成し、回答データを格納・分析するためのスプレッドシートも自動的にセットアップするものです。

スクリプトの主な機能要件

1. メイン機能

  • Java Silver模擬試験用のGoogle Formを自動生成
  • 回答結果を保存・分析するためのスプレッドシートを自動作成
  • フォームとスプレッドシートを連携させる
  • カテゴリ別の分析機能をスプレッドシートに実装

2. Google Formの詳細仕様

  • 試験名：「Java Silver模擬テスト①」
  • 問題数：50問（選択式）
  • 試験時間：75分（タイマー設定）
  • 配点：1問2点（合計100点満点）
  • 合格ライン：70点以上
  • テーマ：プロフェッショナルなブルーベースのテーマ

3. 受験者情報の収集

  • 受験者の名前（必須）
  • メールアドレス（必須）

4. 問題構成（カテゴリ別）

  • 基本構文と変数の宣言：10問
  • 基本データ型と参照型：10問
  • 演算子と式の評価：10問
  • 制御構文（if-else, switch, for, while）：10問
  • 配列の宣言と操作：10問
  • 難易度分布：基本35問、中級10問、難問5問（難問にはマーキング）

5. スプレッドシートの構成

  • メインシート：全回答データ（採点結果含む）
  • 分析シート：合否状況、平均点、最高点、最低点
  • カテゴリ別分析：各カテゴリごとの正答率分析
  • 問題別分析：問題ごとの正答率、難易度評価
  • 個人成績表：個別受験者の成績レポート

スクリプトに含める問題例

以下は各カテゴリの問題例です。これらの形式に沿って50問分の問題をスクリプトに含めてください。 各問題には問題文、選択肢（A～E）、正解、解説を含めてください。

【基本構文と変数の宣言】問題例：

問題：次のうち、有効な変数名はどれですか？

A. 123abc

B. _value

C. public

D. class

E. break-point

正解：B

解説：変数名は文字、ドル記号($)、アンダースコア(_)で始まる必要があります。数字で始めることはできず、予約語（public, class, breakなど）は使用できません。また、ハイフン(-)も変数名に使用できません。

【基本データ型と参照型】問題例：

問題：次のうちプリミティブ型ではないものはどれですか？

A. int

B. boolean

C. String

D. char

E. float

正解：C

解説：String はプリミティブ型ではなく、参照型（クラス）です。Javaの8つのプリミティブ型は byte, short, int, long, float, double, char, boolean です。

【演算子と式の評価】問題例：

問題：次のコードを実行した場合、出力される結果はどれですか？

int a = 10;
int b = 3;
System.out.println(a / b);

A. 3

B. 3.0

C. 3.33

D. 3.333333

E. コンパイルエラー

正解：A

解説：Java では整数同士の除算は整数結果になります。10 ÷ 3 = 3.33... ですが、小数部分は切り捨てられて 3 になります。

【制御構文】問題例：

問題：次のループの実行結果として正しいものはどれですか？

for(int i = 0; i < 5; i++) {
    if(i == 3) {
        continue;
    }
    System.out.print(i);
}

A. 0123

B. 01234

C. 0124

D. 012

E. 1234

正解：C

解説：i が 3 のとき、continue 文によってループの残りの部分がスキップされ、次の繰り返しに進みます。そのため、3 は出力されず、0, 1, 2, 4 が出力されます。

【配列の宣言と操作】問題例：

問題：配列の宣言と初期化で正しいものはどれですか？

A. int array = new int[3];

B. int[] array = {1, 2, 3};

C. int array[] = new int[3]{1, 2, 3};

D. int[] array = new int(3);

E. array[] = {1, 2, 3};

正解：B

解説：選択肢B「int[] array = {1, 2, 3};」は配列の宣言と初期化を正しく行っています。Cは構文エラーで、サイズ指定と初期化を同時に行うことはできません。Dは括弧が角括弧ではありません。Eは型宣言がありません。

コード要件

1. スクリプト構造

  • メイン関数: createJavaSilverExam()
  • フォーム作成関数: createExamForm()
  • 問題追加関数: addQuestions(form)
  • スプレッドシート設定関数: setupSpreadsheet(form)
  • 分析機能設定関数: setupAnalytics(sheet)

2. コードスタイル

  • 適切なコメント記述
  • エラーハンドリング処理
  • 変数名は意味が分かりやすいものを使用
  • モジュール化された関数設計

3. 問題データ構造

  • 問題をJSONまたはオブジェクト配列形式で管理
  • 各問題に以下のプロパティを含める:
    • category: カテゴリ名
    • difficulty: 難易度（'basic', 'intermediate', 'advanced'）
    • question: 問題文
    • code: コードスニペット（必要な場合）
    • options: 選択肢の配列
    • answer: 正解のインデックスまたは正解文字列
    • explanation: 解説文

実行方法・使用手順

  1. Google ドライブで新しいGoogle Apps Scriptファイルを作成
  2. 生成されたコードをコピーして貼り付け
  3. 必要に応じて、メール通知や共有設定などをカスタマイズ
  4. 「実行」ボタンをクリックしてスクリプトを実行
  5. 必要な権限を許可
  6. 生成されたフォームとスプレッドシートへのリンクがログに表示される

スクリプト実装例（一部）

以下は、メイン関数とフォーム作成関数の実装例です。これを参考に完全なスクリプトを作成してください。

/**
* Java Silver模擬試験のフォームとスプレッドシートを生成する
* メイン関数
*/
function createJavaSilverExam() {
  try {
    // フォームの作成
    const form = createExamForm();

    // 問題の追加
    addQuestions(form);

    // スプレッドシートの設定
    const sheet = setupSpreadsheet(form);

    // 分析機能の設定
    setupAnalytics(sheet);

    Logger.log('模擬試験の作成が完了しました。');
    Logger.log('フォームURL: ' + form.getPublishedUrl());
    Logger.log('スプレッドシートURL: ' + sheet.getUrl());

    return {
      form: form,
      sheet: sheet
    };
  } catch (error) {
    Logger.log('エラーが発生しました: ' + error);
    return null;
  }
}

/**
* 試験用フォームを作成する
* @return {FormApp.Form} 作成されたフォーム
*/
function createExamForm() {
  // フォームの作成と基本設定
  const form = FormApp.create('Java Silver模擬テスト①');
  form.setDescription('Java SE 11 Silver試験対策のための模擬テストです。全50問、制限時間75分、合格ラインは70点以上（100点満点）です。すべての問題に回答してください。');
  form.setIsQuiz(true);
  form.setLimitOneResponsePerUser(true);
  form.setCollectEmail(true);
  form.setProgressBar(true);
  form.setPublishingSummary(true);

  // テーマの設定（青系）
  // 注：テーマのIDはAPI制限があるため、実際のコードでは別の方法が必要

  // 制限時間の設定（Google FormsのAPIではサポートされていないため注釈）
  // 実際には説明文に制限時間を記載することで対応

  // 受験者情報セクション
  const infoSection = form.addPageBreakItem();
  infoSection.setTitle('受験者情報');

  // 名前
  const nameItem = form.addTextItem();
  nameItem.setTitle('氏名');
  nameItem.setRequired(true);

  // メールアドレス（フォーム設定で収集するため不要）

  // 試験説明セクション
  const examSection = form.addPageBreakItem();
  examSection.setTitle('試験説明');
  examSection.setHelpText('このテストは以下のカテゴリから出題されます：\n' +
    '• 基本構文と変数の宣言\n' +
    '• 基本データ型と参照型\n' +
    '• 演算子と式の評価\n' +
    '• 制御構文（if-else, switch, for, while）\n' +
    '• 配列の宣言と操作\n\n' +
    '試験時間は75分、全50問（各2点）で100点満点です。70点以上で合格です。');

  return form;
}

注意事項：

  • 実際のGoogle Apps ScriptではGoogle FormsのAPIの制限があるため、一部の機能（例：制限時間の設定）は直接実装できない場合があります。その場合は代替手段を提示してください。
  • 50問すべての詳細な問題と回答を含めてください。これによりスクリプトが1回の実行で完全な模擬テストを生成できるようにします。
  • スプレッドシートの分析機能は、数式やチャートを使用して視覚的に理解しやすいものにしてください。
