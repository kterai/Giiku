月間概要

Java SE 17 Programmer資格取得に必要な知識と技術を習得し、実践的なJavaプログラミングスキルを身につけます。Javaの基礎から応用、オブジェクト指向の概念まですべて体系的に学びます。

達成目標

  • Java SE 17の言語仕様と基本APIを理解する
  • オブジェクト指向プログラミングの概念と実装方法を習得する
  • Java Silver試験の出題範囲を網羅する
  • 実践的なJavaプログラムを作成できるようになる

習得スキル

Java構文・プログラミング技法 オブジェクト指向3大要素 例外処理とリソース管理 コレクションAPIとStream API Java SE 17新機能
