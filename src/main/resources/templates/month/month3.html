第3ヶ月目: Webアプリケーション開発実践

フロントエンドとバックエンドの技術を組み合わせ、実用的なWebアプリケーションを開発します

月間概要

月間学習目標

Webアプリケーション開発に必要なフロントエンドとバックエンドの技術を習得し、実際にWebアプリケーションを設計・開発する能力を身につけます。HTML/CSS/JavaScriptによるフロントエンド開発から、Spring Bootを使ったバックエンド開発、データベース連携まで、Webアプリケーション開発の全体像を理解し実践します。

達成目標

  • HTML/CSS/JavaScriptを使ったフロントエンド開発ができる
  • PostgreSQLでデータベース設計と操作ができる
  • Spring Bootを使ったバックエンド開発ができる
  • フロントエンドとバックエンドを連携させたWebアプリケーションを開発できる
  • 開発したアプリケーションをポートフォリオとして提示できる

学習期間

18日間

Day 37 - Day 54

習得スキル

HTML5 CSS3 JavaScript Bootstrap PostgreSQL Spring Boot REST API Git/GitHub テスト
