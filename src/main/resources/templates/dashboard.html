育成目標

このカリキュラムは、IT業界未経験者を3ヶ月間で基本的なITエンジニアスキルを習得させ、実際の開発現場で活躍できる人材を育成することを目指しています。段階的な学習設計により、プログラミングの基礎から実践的なWeb開発まで、体系的に学習できる内容となっています。

段階的学習

基礎から応用まで、無理のないステップで確実にスキルアップできます。

資格取得サポート

Java Silver、基本情報技術者試験の合格を目指した実践的なカリキュラム。

実践重視

理論だけでなく、実際のWebアプリケーション開発を通じて技術を習得。

対象者

20代～30代の未経験者

Windows・Microsoft Officeの基本操作ができるレベルからスタート。IT業界への転職やキャリアチェンジを希望する方に最適です。
