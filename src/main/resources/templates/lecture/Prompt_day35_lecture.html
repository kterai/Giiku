2.1 基本情報技術者試験制度の詳細解説（60分）

午前試験詳細分析
  • • 出題形式：四肢択一式80問
  • • 試験時間：150分（1問1.875分）
  • • 合格基準：60点以上（48問以上正解）

  • • マネジメント系：20問（25%）
  • • ストラテジ系：10問（12.5%）
午後試験詳細分析
  • • 出題形式：記述式11問中5問選択
  • • 試験時間：150分（1問30分）
  • • 合格基準：60点以上
  • • 必須：情報セキュリティ・アルゴリズム
  • • 選択：プログラミング・DB・NW

講師向け指導ポイント： 受験者には試験時間の体感を理解させるため、実際にストップウォッチを使用して1問あたりの時間を測定させる。 特に午前試験では、最初の30分で全問を一度見て難易度を把握する戦略を具体的に指導する。

午前試験時間配分戦略
時間帯 活動内容 目標 注意点
0-30分 全問題の概観・分類 問題難易度の把握 解答せず、マーキングのみ

2.4 計算問題の解法パターン体系化（60分）

頻出計算問題パターン集
パターン1 サブネット計算

基本公式：

ホスト数 = 2^(32-プレフィックス長) - 2

サブネット数 = 2^(借用ビット数)

ブロードキャストアドレス = ネットワークアドレス + ホスト数 + 1

例題：

192.168.1.0/26のホスト数は？

答：2^(32-26) - 2 = 62個

指導ポイント： 2の累乗計算を暗記させる（2^1=2, 2^2=4, ..., 2^8=256）。ビット演算の可視化で理解促進。

パターン2 CPU性能・処理能力計算

基本公式：

処理時間 = 命令数 ÷ (クロック周波数 ÷ CPI)

MIPS値 = クロック周波数(MHz) ÷ CPI

スループット = 1 ÷ 処理時間

例題：

100MIPS、命令数10^6の処理時間は？

答：10^6 ÷ (100×10^6) = 0.01秒

パターン3 可用性・信頼性計算

基本公式：

直列システム可用性 = A × B

並列システム可用性 = 1 - (1-A) × (1-B)

MTBF = 稼働時間 ÷ 故障回数

例題：

可用性0.9の機器2台並列の可用性は？

答：1-(1-0.9)^2 = 0.99

// 回答表示機能
function toggleAnswer(questionId) {
    const answerDiv = document.getElementById(questionId + '-answer');
    const button = event.target;
    
    if (answerDiv.classList.contains('hidden')) {
        answerDiv.classList.remove('hidden');
        button.innerHTML = '<i class="fas fa-eye-slash mr-2"></i>回答を隠す';
        button.classList.add('bg-gray-500', 'hover:bg-gray-600');
        button.classList.remove('bg-blue-500', 'hover:bg-blue-600');
    } else {
        answerDiv.classList.add('hidden');
        button.innerHTML = '<i class="fas fa-eye mr-2"></i>回答を表示';
        button.classList.add('bg-blue-500', 'hover:bg-blue-600');
        button.classList.remove('bg-gray-500', 'hover:bg-gray-600');
    }
}

// 試験タイマー機能
function startExamTimer(minutes) {
    let timeLeft = minutes * 60;
    const timerDisplay = document.getElementById('timer-display');
    
    const timer = setInterval(() => {
        const hrs = Math.floor(timeLeft / 3600);
        const mins = Math.floor((timeLeft % 3600) / 60);
        const secs = timeLeft % 60;
        
        timerDisplay.textContent = 
            `${hrs.toString().padStart(2, '0')}:${mins.toString().padStart(2, '0')}:${secs.toString().padStart(2, '0')}`;
        
        if (timeLeft <= 0) {
            clearInterval(timer);
            alert('試験時間終了です！');
        }
        timeLeft--;
    }, 1000);
}
