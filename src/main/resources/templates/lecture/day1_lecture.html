1. Java言語の特徴と歴史

講師向けポイント

Javaの「Write Once, Run Anywhere（一度書けばどこでも動く）」という特徴を、身近な例（スマートフォンアプリが異なる機種で動く）で説明し、なぜ企業システム開発で重宝されるかを具体的に伝えましょう。

Javaの主要特徴

特徴 詳細説明 実際のメリット
プラットフォーム独立
（Platform Independent） 		JVM（Java仮想マシン：Javaコードを実行する環境）上で動作するため、Windows、Mac、Linuxなど異なるOS上で同じプログラムが動作 銀行システムをWindows開発→Linuxサーバーで運用が可能。開発コスト大幅削減
オブジェクト指向
（Object-Oriented） 		現実世界のモノ（オブジェクト）の概念でプログラムを設計。カプセル化（データ保護）、継承（機能拡張）、ポリモーフィズム（柔軟な処理）を実現 「顧客」クラスを作成→「法人顧客」「個人顧客」に拡張可能。大規模システムの保守性向上
メモリ自動管理
（Garbage Collection） 		ガベージコレクション（不要メモリ自動回収機能）により、プログラマーがメモリ管理を意識する必要がない メモリリーク（メモリ不足）によるシステムクラッシュを防止。24時間稼働システムの安定性確保
豊富なライブラリ
（Rich Standard Library） 		標準API（アプリケーションプログラミングインターフェース：機能を呼び出すための仕組み）が充実。データベース接続、ネットワーク通信、GUI作成などが標準で利用可能 ECサイト開発で決済処理、在庫管理、ユーザー認証などの機能を短期間で実装可能

日常例での理解

Javaは「翻訳アプリ」のような存在です。あなたが日本語で書いた文章（Javaコード）を、世界中のどの国の人（Windows、Mac、Linux）でも理解できる共通言語（バイトコード）に翻訳してくれます。そのため、一度プログラムを作れば、どんなコンピュータでも動かすことができるのです。

2. 開発環境構築

講師向け重要ポイント

環境構築は初心者が最も躓きやすい箇所です。各手順で「なぜこの作業が必要なのか」を説明し、エラーが発生した場合の対処法も事前に準備しておきましょう。特にPATH設定（環境変数）の概念は丁寧に説明してください。

必要なツールの概要

JDK（Java Development Kit）

正式名称：Java開発キット

役割：Javaプログラムを作成・実行するために必要なツール一式

比喩：料理を作るための基本的な調理器具セット（包丁、まな板、鍋など）

IDE（Integrated Development Environment）

正式名称：統合開発環境

役割：コード作成、デバッグ、実行を効率化するツール

比喩：料理の際の電子レンジやフードプロセッサー（作業を自動化・効率化）

4. 演習問題

演習問題1: Javaの特徴理解

以下の質問に答えてください：

  1. 1. Javaが「Write Once, Run Anywhere」と呼ばれる理由を、具体例を交えて説明してください。
  2. 2. JDK、JRE、JVMの違いと役割を説明してください。
  3. 3. なぜJavaでメモリ管理を意識する必要が少ないのか説明してください。

回答例：

1. 「Write Once, Run Anywhere」の理由：

Javaコードは一度コンパイルするとバイトコードという中間言語に変換され、JVM（Java仮想マシン）上で実行される。JVMが各OS（Windows、Mac、Linux）に対応しているため、同じバイトコードがどのプラットフォームでも動作する。例：銀行のオンラインバンキングシステムをWindows環境で開発し、LinuxサーバーやMacの管理画面で同じプログラムを動かすことができる。

2. JDK、JRE、JVMの違い：

  • • JVM（Java Virtual Machine）：Javaプログラムを実行する仮想マシン
  • • JRE（Java Runtime Environment）：JVM + 実行に必要なライブラリ
  • • JDK（Java Development Kit）：JRE + 開発ツール（コンパイラなど）

3. メモリ管理が不要な理由：

Javaにはガベージコレクション（GC）機能があり、使用されなくなったオブジェクトを自動的にメモリから削除する。これにより、プログラマーが手動でメモリを解放する必要がなく、メモリリークによるシステムクラッシュを防げる。

演習問題2: Hello Worldの改良

以下の要件を満たすプログラムを作成してください：

  • • 自分の名前を表示する
  • • 今日の日付を表示する
  • • 好きなプログラミング言語を表示する
  • • 罫線（*印）で出力を装飾する

回答例：

public class MyProfile {
    public static void main(String[] args) {
        // 罫線で装飾
        System.out.println("*************************");
        System.out.println("*    自己紹介プログラム    *");
        System.out.println("*************************");
        
        // 個人情報の表示
        System.out.println("名前: 田中 太郎");
        System.out.println("日付: 2025年8月3日");
        System.out.println("好きな言語: Java");
        
        System.out.println("*************************");
        System.out.println("* Java学習を頑張ります！ *");
        System.out.println("*************************");
    }
}

ポイント：複数のSystem.out.println()を使用して、見やすい形式で情報を表示しています。

演習問題3: エラー修正

以下のコードにはいくつかのエラーがあります。エラーを見つけて修正してください：

public class helloworld {
    public static void Main(String[] args) {
        system.out.println("Hello World")
        System.out.println("Welcome to Java!)
    }

修正版コード：

public class HelloWorld {  // クラス名は大文字で開始
    public static void main(String[] args) {  // mainは小文字
        System.out.println("Hello World");  // Systemは大文字、セミコロン必須
        System.out.println("Welcome to Java!");  // 閉じダブルクォート修正
    }
}

エラー箇所の解説：

  • • クラス名：「helloworld」→「HelloWorld」（大文字で開始、キャメルケース）
  • • mainメソッド：「Main」→「main」（小文字）
  • • System：「system」→「System」（大文字で開始）
  • • セミコロン：文の最後に「;」が必要
  • • 文字列：ダブルクォートの閉じ忘れ

5. 理解度クイズ

クイズ1: Javaの特徴

Javaの主要な特徴として正しいものはどれですか？

正解: C) Write Once, Run Anywhereを実現している

解説：JavaはJVM上で動作するため、一度コンパイルしたプログラムが異なるプラットフォーム（Windows、Mac、Linux）で実行可能です。これがJavaの最大の特徴の一つです。

クイズ2: コンパイルプロセス

Javaプログラムの実行プロセスで正しい順序はどれですか？

正解: B) .java → javac → .class → JVM → 実行結果

解説：Javaソースファイル(.java)をjavacコンパイラでバイトコード(.class)に変換し、JVMがそれを実行して結果を出力します。この2段階プロセスがJavaの特徴です。

クイズ3: Hello Worldコード

以下のコードで間違っている箇所はどれですか？

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

正解: D) コードに間違いはない

解説：このコードは完全に正しいHello Worldプログラムです。クラス名、mainメソッドの宣言、System.out.println文、すべて正しく記述されています。
