局所性の原理

時間的局所性

最近使用されたデータは再び使用される可能性が高い

例：ループ処理での変数参照

空間的局所性

近隣のアドレスのデータが連続して使用される

例：配列の順次アクセス

キャッシュヒット率と性能

実効アクセス時間 =
ヒット率 × キャッシュアクセス時間
+ ミス率 × 主記憶アクセス時間
例：ヒット率90%、キャッシュ2ns、主記憶20ns
→ 0.9×2 + 0.1×20 = 3.8ns

4. 入出力システムとインターフェース

入出力制御方式

プログラム制御方式

CPUが直接I/Oデバイスを制御

特徴：
• シンプルな構成
• CPUの処理能力を消費
• 低速なI/Oに適用

特徴：
• CPUの有効活用

• 高速データ転送

演習問題

問題1: CPU性能計算

あるCPUの仕様が以下の通りである時、MIPSを求めなさい。

  • • クロック周波数：2.4GHz
  • • 平均CPI：3

※ MIPS = クロック周波数 ÷ CPI ÷ 10⁶

解答・解説

計算過程：

MIPS = 2.4 × 10⁹ ÷ 3 ÷ 10⁶

= 2.4 × 10³

= 800 MIPS

解説：

1秒間に実行できる命令数は、クロック周波数をCPIで割ったものになります。この場合、1秒間に8億命令を実行できることを意味します。

問題2: キャッシュヒット率と性能

以下の条件でのキャッシュシステムの実効アクセス時間を求めなさい。

  • • キャッシュアクセス時間：2ns
  • • 主記憶アクセス時間：20ns
  • • キャッシュヒット率：95%

解答・解説

計算過程：

実効アクセス時間 = ヒット率 × キャッシュ時間 + ミス率 × 主記憶時間

= 0.95 × 2 + 0.05 × 20

= 1.9 + 1.0

= 2.9ns

解説：

キャッシュヒット率が95%の場合、95%の確率で2nsでアクセスでき、5%の確率で20nsかかります。平均すると2.9nsでアクセスできることになります。

問題3: 記憶装置選択

以下の要件を満たす記憶装置として最適なものを選び、理由を説明しなさい。

  • • 容量：10TB以上
  • • コスト：できるだけ安価
  • • 用途：動画ファイルのバックアップ保存
  • • アクセス頻度：月1回程度

解答・解説

推奨：HDD（ハードディスクドライブ）

選択理由：

  • 大容量（10TB以上）の製品が豊富
  • GB当たりの単価が最も安価
  • アクセス頻度が低いため、速度の遅さは問題なし
  • 動画ファイルは連続読み取りが多く、HDDの特性に適合

補足：

SSDは高速だが高価で、この用途には過剰。光ディスクは容量が不足。フラッシュメモリは中間的だが、この容量では高コスト。

理解度クイズ

クイズ1: CPUの構成要素

CPUの基本構成要素として正しい組み合わせはどれか。

正解：B. 制御装置・演算装置・レジスタ

解説：

CPUの基本構成要素は、制御装置（CU）、演算装置（ALU）、レジスタの3つです。メモリやハードディスクは外部装置、キャッシュは高速化のための補助装置です。

クイズ2: メモリ階層

メモリ階層を速度の速い順に並べたものはどれか。

正解：A. レジスタ → キャッシュ → 主記憶 → 補助記憶

解説：

メモリ階層は、レジスタが最も高速（1クロック）、次にキャッシュ（数クロック）、主記憶（数十ns）、補助記憶（ms級）の順になります。一般に、速度が速いほど容量は小さく、コストは高くなります。

クイズ3: 入出力制御方式

DMA（Direct Memory Access）制御方式の特徴として正しいものはどれか。

正解：C. CPUを介さずにメモリとI/Oデバイス間でデータ転送する

解説：

DMA制御方式では、DMAコントローラがメモリとI/Oデバイス間の直接データ転送を制御します。これによりCPUの負荷を軽減し、高速なデータ転送が可能になります。転送完了時に割り込みでCPUに通知します。
