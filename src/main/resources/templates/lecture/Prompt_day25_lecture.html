メイン講義内容の改良指針

1. ルータ・スイッチ（改良ポイント）

分かりやすい表現

  • • L2スイッチ = 「同じフロアの郵便配達員」
  • • L3ルータ = 「異なる都市間の宅配便」
  • • MACアドレス = 「住民票の番号」
  • • IPアドレス = 「住所」

実務での判断基準

  • • 同一VLAN内 → L2スイッチ使用
  • • 異なるネットワーク間 → L3ルータ使用
  • • コスト重視 → L2スイッチ
  • • セキュリティ重視 → L3ルータ

2. ルーティングプロトコル（比較表改良）

プロトコル 特徴（日本語説明） 適用場面 メリット・デメリット
RIP
（距離ベクトル型）		 ホップ数で最短経路を決定
最大15ホップまで		 小規模ネットワーク
設定が簡単		 ○簡単設定
×収束が遅い
OSPF
（リンクステート型）		 帯域幅を考慮した最適経路
エリア分割可能		 中〜大規模ネットワーク
企業内ネットワーク		 ○高速収束
×設定が複雑

実務での選択基準:

  • • 拠点数10以下 → RIP で十分
  • • 拠点数10-100 → OSPF を推奨
  • • インターネット接続 → BGP が必要

3. VLAN（概念の視覚化）

VLANの概念（アナロジー）

物理的な例え:

  • • 1つのビル（スイッチ）
  • • 複数の会社（VLAN）が入居
  • • 会社ごとに独立したネットワーク
  • • エレベーター（トランクポート）で会社間移動

VLAN設定例

// 管理部門用VLAN
VLAN 10: 192.168.10.0/24
// 開発部門用VLAN
VLAN 20: 192.168.20.0/24
// ゲスト用VLAN
VLAN 99: 192.168.99.0/24

ネットワーク応用技術 - 専門用語解説強化指針

必須解説用語（例）

VLAN (Virtual LAN)

→ 仮想LAN。物理的に離れた機器を論理的に同じネットワークにまとめる技術

トランクポート

→ 複数のVLAN情報を1つのケーブルで運ぶ仕組み。「情報の荷札」付きで配送

STP (Spanning Tree Protocol)

→ ループ防止プロトコル。道路の「一方通行」のような仕組み
