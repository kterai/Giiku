  • • Text Blocks → 「テキストブロック（複数行文字列の記述簡略化）」
  • • Records → 「レコード（データホルダークラスの自動生成）」

具体的なコード例を必須追加

// Pattern Matching for instanceof（改善前後比較）
// 改善前（Java16以前）
if (obj instanceof String) {
    String str = (String) obj;
    System.out.println(str.length());
}

// 改善後（Java17）
if (obj instanceof String str) {
    System.out.println(str.length()); // 自動キャスト
}

実用例を具体的に記載
