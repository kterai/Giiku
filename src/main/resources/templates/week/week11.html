第11週: Spring Boot開発実践

実務レベルのSpring Boot開発技術を習得し、フロントエンドとの連携まで学ぶ5日間

週間概要

この週では、Spring Bootを使った実践的なWebアプリケーション開発技術を学びます。Spring Data JPAによるデータアクセス層の実装、Spring Securityによる認証・認可の仕組み、MVC設計に基づいたWebアプリ開発、テスト駆動開発とユニットテスト、そしてフロントエンドとバックエンドの連携など、実務で即戦力として活躍できるレベルの技術を習得します。

週間目標

  • Spring Data JPAを使用したデータベースアクセス層の実装ができる
  • Spring Securityによる認証・認可の仕組みを理解し実装できる
  • 実践的なWebアプリケーション開発（フォーム処理、バリデーション、REST API）ができる
  • JUnitを使用したテストコードの作成と品質管理ができる
  • フロントエンド（JavaScript）とバックエンド（Spring Boot）の連携方法を理解する

使用技術スタック

Spring Boot Spring Data JPA Spring Security Spring MVC PostgreSQL JUnit 5 Mockito JavaScript Fetch API REST API
