週間概要

基本情報技術者試験の最終週として、ストラテジ系分野（経営戦略、システム戦略、法務・知財）の学習と、 これまで2ヶ月間で学んだ全分野の総合対策を実施します。本格的な模擬試験を通じて実戦力を高め、 試験合格に向けた最終準備を完成させます。

ストラテジ系学習

経営戦略、システム戦略、法務・知財の重要ポイントを効率的に習得します。

総合模擬試験

全分野を網羅した本格的な模擬試験で実戦力を確認・向上させます。

弱点強化

個別の弱点を特定し、集中的な対策で合格ラインを確実に超えます。

週間学習目標

この週の学習目標

  • ストラテジ系分野の基本概念と重要ポイントを理解し、試験問題に対応できるようになる
  • 経営戦略、システム戦略、法務・知財の関連性を理解し、体系的に整理できるようになる
  • 本格的な模擬試験を通じて実戦感覚を身につけ、時間配分と解答戦略を確立する
  • 全分野の弱点を特定し、集中的な復習で知識の定着と応用力向上を図る

ストラテジ系の主要概念

企業活動と法務

主な特徴:

  • • 組織形態と企業活動の理解
  • • 法務・コンプライアンスの基本
  • • 労働関連法規の知識
  • • 契約・取引の法的側面

経営戦略

主な特徴:

  • • 経営戦略の基本概念と手法
  • • SWOT分析、ポートフォリオ分析
  • • マーケティング戦略の基本
  • • 競争戦略と差別化

システム戦略

主な特徴:

  • • 情報システム戦略の立案
  • • IT投資効果と評価
  • • システム企画と要件定義
  • • 調達・外部委託の管理

知的財産権

主な特徴:

  • • 著作権・特許権の基本
  • • ソフトウェアライセンス
  • • 標準化と規格の重要性
  • • 国際標準化機関の役割

試験対策戦略

時間配分戦略

  • • 午前試験: 80問/150分
  • • 1問あたり約1.9分を目安
  • • 見直し時間を必ず確保
  • • 難問は後回しにする

分野別重要度

  • • テクノロジ系: 50問（約62%）
  • • マネジメント系: 10問（約13%）
  • • ストラテジ系: 20問（約25%）
  • • 全分野で60%以上が目標

解答テクニック

  • • 選択肢を先に読む
  • • 消去法を活用する
  • • 計算問題は概算で判断
  • • 迷った問題はマークして後で
