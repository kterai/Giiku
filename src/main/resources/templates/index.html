育成目標

このカリキュラムは、IT業界未経験者を3ヶ月間で基本的なITエンジニアスキルを習得させ、実際の開発現場で活躍できる人材を育成することを目指しています。段階的な学習設計により、プログラミングの基礎から実践的なWeb開発まで、体系的に学習できる内容となっています。

段階的学習

基礎から応用まで、無理のないステップで確実にスキルアップできます。

資格取得サポート

Java Silver、基本情報技術者試験の合格を目指した実践的なカリキュラム。

実践重視

理論だけでなく、実際のWebアプリケーション開発を通じて技術を習得。

対象者

20代～30代の未経験者

Windows・Microsoft Officeの基本操作ができるレベルからスタート。IT業界への転職やキャリアチェンジを希望する方に最適です。

プログラム構成

本カリキュラムは、以下の3ヶ月に分けて設計されています。各月は18日間（約4週間）で構成されており、それぞれ明確な目標を持っています。

第1ヶ月目

Java Silver 17
目標: Oracle Certified Professional Java SE 17 Developer資格取得レベルの知識習得
主な学習内容:
  • Java言語基礎・基本構文
  • オブジェクト指向プログラミング
  • Java言語応用（コレクション、Stream API等）
  • Java SE 17の新機能
詳細を見る

第2ヶ月目

基本情報技術者
目標: 基本情報技術者試験合格レベルのIT基礎知識習得
主な学習内容:
  • コンピュータシステム・ハードウェア
  • ネットワーク・データベース
  • アルゴリズム・セキュリティ
  • プロジェクトマネジメント・経営戦略
詳細を見る

第3ヶ月目

Web開発
目標: 実用的なWebアプリケーション開発スキルの習得
主な学習内容:
  • HTML/CSS/JavaScript基礎
  • PostgreSQLとサーバーサイド開発
  • Spring Boot実践
  • 総合開発プロジェクト
