Day 22: ディジタル回路・論理設計

論理回路、論理演算、ブール代数の基礎を学び、コンピュータの動作基盤を理解する

第2ヶ月目 第5週 22日目

学習目標

  • カルノー図を用いて最小項表現を求められる

ディジタル回路の基礎

基本論理ゲート

ディジタル回路の基本構成要素である論理ゲートについて詳しく学習します。各ゲートは特定の論理演算を実行し、コンピュータ内部の情報処理の基盤となります。

ANDゲート

すべての入力が1（HIGH）の時のみ出力が1になる論理積演算を行います。

記号: A・B または A∧B
すべてが真の時のみ真
A B 出力
000
010
100
111

ORゲート

いずれかの入力が1（HIGH）の時に出力が1になる論理和演算を行います。

記号: A+B または A∨B
いずれかが真なら真
A B 出力
000
011
101
111

ブール代数の基本法則

論理回路の設計と最適化に必要なブール代数の重要な法則について学習します。

基本法則
  • 交換法則: A + B = B + A
  • 結合法則: (A + B) + C = A + (B + C)
  • 分配法則: A(B + C) = AB + AC
  • 恒等法則: A + 0 = A, A・1 = A
重要な定理
  • 補数法則: A + A' = 1, A・A' = 0
  • 吸収法則: A + AB = A
  • ド・モルガンの定理: (A + B)' = A'・B'
  • 二重否定: (A')' = A

組合せ回路と順序回路

組合せ回路

現在の入力のみで出力が決まる回路

  • 加算器（半加算器、全加算器）
  • デコーダ（3-to-8デコーダなど）
  • エンコーダ（8-to-3エンコーダなど）
  • マルチプレクサ、デマルチプレクサ
順序回路

過去の状態を記憶する回路

  • フリップフロップ（RS、JK、D、T）
  • カウンタ（同期、非同期）
  • レジスタ（シフトレジスタなど）
  • 状態機械（有限状態機械）
