10:15-11:00
午前講義パート1: 条件分岐 (if-else, switch)

Javaの条件分岐構文について学びます。if文、if-else文、else-if文、switch文の使い方と適切な使用場面について解説します。

// if-else文の例
if (score >= 80) {
    System.out.println("優");
} else if (score >= 70) {
    System.out.println("良");
} else if (score >= 60) {
    System.out.println("可");
} else {
    System.out.println("不可");
}

// switch文の例
switch (day) {
    case 1:
        System.out.println("月曜日");
        break;
    case 2:
        System.out.println("火曜日");
        break;
    // ... 他の曜日 ...
    default:
        System.out.println("無効な日付");
}

午後講義パート1: 繰り返し処理 (for, while, do-while)

Javaの繰り返し処理構文について学びます。for文、while文、do-while文の違いと使い方、break文とcontinue文の役割について解説します。

// for文の例
for (int i = 0; i < 5; i++) {
    System.out.println("カウント: " + i);
}

// while文の例
int i = 0;
while (i < 5) {
    System.out.println("カウント: " + i);
    i++;
}

// do-while文の例
int j = 0;
do {
    System.out.println("最低1回は実行: " + j);
    j++;
} while (j < 5);

// 配列の宣言と初期化

numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;

// 配列の要素へのアクセス
System.out.println("3番目の得点: " + scores[2]);  // 78

テーマ詳細

条件分岐構文

条件分岐はプログラムの流れを制御する基本的な構文です。条件に応じて異なる処理を行いたい場合に使用します。

if-else文

条件が真の場合と偽の場合で処理を分けます。複数の条件を連鎖させる場合はelse-if文を使用します。

if (条件1) {
    // 条件1が真の場合の処理
} else if (条件2) {
    // 条件1が偽で条件2が真の場合の処理
} else {
    // すべての条件が偽の場合の処理
}

switch文

ある変数の値に応じて処理を分ける場合に使用します。複数の同じ値に対して同じ処理を行う場合に便利です。

switch (変数) {
    case 値1:
        // 変数が値1の場合の処理
        break;
    case 値2:
    case 値3:
        // 変数が値2または値3の場合の処理
        break;
    default:
        // どの値にも一致しない場合の処理
}

※ switchでは整数型、文字型、文字列型（Java 7以降）のみ使用可能です。

繰り返し処理構文

繰り返し処理は同じ処理を複数回実行する場合に使用します。Javaでは3種類の繰り返し構文があります。

for文

繰り返し回数が決まっている場合に使用します。初期化、条件、更新の3つの部分で構成されます。

for (初期化; 条件; 更新) {
    // 繰り返し実行される処理
}

while文

条件が真の間、処理を繰り返し実行します。条件が最初から偽の場合は一度も実行されません。

while (条件) {
    // 繰り返し実行される処理
}

do-while文

処理を実行してから条件を評価します。最低1回は処理が実行されることが保証されます。

do {
    // 繰り返し実行される処理
} while (条件);

break文とcontinue文

break文: ループを即座に終了します。
continue文: 現在の繰り返しをスキップし、次の繰り返しに進みます。

配列

配列は同じ型の複数の値をまとめて管理するためのデータ構造です。インデックスを使用して各要素にアクセスします。

配列の宣言と初期化

// 方法1: サイズを指定して宣言後、個別に要素を初期化
int[] array1 = new int[5];
array1[0] = 10;
array1[1] = 20;
// ...

// 方法2: 宣言と同時に初期化
int[] array2 = {10, 20, 30, 40, 50};

// 方法3: 宣言と初期化を別々に
int[] array3;
array3 = new int[]{10, 20, 30, 40, 50};

配列の操作

要素へのアクセス: インデックス（0から始まる）を使用してアクセスします。
配列の長さ: length属性で取得します。
配列の走査: for文やfor-each文（拡張for文）を使用します。

// for文による配列の走査
for (int i = 0; i < array.length; i++) {
    System.out.println(array[i]);
}

// 拡張for文による配列の走査
for (int value : array) {
    System.out.println(value);
}
注意点: インデックスは0から始まります。配列の範囲外にアクセスすると「ArrayIndexOutOfBoundsException」が発生します。
