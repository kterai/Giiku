ITエンジニア育成カリキュラム

次のDayのスライド作成手順ガイド

はじめに

このガイドでは、ITエンジニア育成カリキュラムの次のDay向けスライドをGammaAIを使って効率的に作成する方法を説明します。Day1のスライドを作成した手順を応用して、Day2以降のスライドも同様の品質で効率的に作成できるようにしましょう。

ポイント

プロンプトテンプレートを活用することで、毎回一から作り直す必要がなく、効率的に高品質なスライドを作成できます。

スライド作成の全体ワークフロー

1

対象Dayの内容を理解する

対象となるDayのHTML内容を確認し、学習目標、達成基準、詳細スケジュール、テーマ詳細などを把握します。

確認すべき項目：

  • 学習目標と達成基準
  • 時間割と各セクションの内容
  • 主要テーマと技術的詳細
  • 演習内容とサンプルコード
  • 理解度確認項目
2

汎用テンプレートを取得する

AIドライブの「/ITエンジニア育成カリキュラム/スライド作成用テンプレート」から汎用テンプレートを取得します。

以下のファイルを参照してください：

  • Day1_スライドプロンプトテンプレート.html - Day1のスライド作成に使用したプロンプト
  • 汎用スライドプロンプトテンプレート.html - 任意のDayに応用できる汎用テンプレート
3

プロンプトをカスタマイズする

取得したテンプレートを、対象Dayの内容に合わせてカスタマイズします。

カスタマイズすべき主要部分：

  • タイトルとサブタイトル（Day番号と内容）
  • 学習目標（対象Dayの目標に合わせる）
  • スケジュール（時間割と各セッションの内容）
  • 各スライドの内容（テーマ詳細、コード例、図解など）
  • 演習内容（対象Dayの演習に合わせる）

例えば、Day2の「データ型と変数」の場合：

### スライド2：本日の学習目標
以下の箇条書きリストを含める:
- Javaの基本データ型と参照型の違いを理解する
- 変数の宣言と初期化の方法をマスターする
- 型変換（キャストとプロモーション）を適切に使用できる
- 各種演算子の優先順位と結合規則を理解する

### スライド7：基本データ型（プリミティブ型）
以下のデータ型を表形式で説明:
- 整数型: byte (8bit), short (16bit), int (32bit), long (64bit)
- 浮動小数点型: float (32bit), double (64bit)
- 文字型: char (16bit)
- 論理型: boolean (1bit)
各型のサイズ、値の範囲、デフォルト値、リテラルの表記法を含める
4

GammaAIでスライドを作成する

カスタマイズしたプロンプトをGammaAIに入力してスライドを作成します。

GammaAIの使用手順：

  1. GammaAI（https://gamma.app/）にアクセス
  2. 新規プレゼンテーションを作成
  3. カスタマイズしたプロンプトを入力
  4. 生成されたスライドを確認し、必要に応じて調整

注意点

生成されたスライド内のコードや技術的内容に誤りがないか必ず確認してください。AIは時々不正確な情報を生成することがあります。

5

スライドの確認と修正

生成されたスライドを確認し、必要に応じて修正を行います。

確認すべき点：

  • 技術的内容の正確さ（特にコード例や説明）
  • スライドの流れと論理的構成
  • 視覚的要素（図解、表、コード表示など）の適切さ
  • 時間配分と各トピックのバランス
  • スペルや文法のエラー
  • IDE（Pleiades）とJDK（Amazon Corretto）への言及の一貫性

効果的なスライド作成のためのベストプラクティス

各Dayの特性を活かす

それぞれのDayには特有のテーマや重点があります。例えば：

  • Day2：データ型と変数 → データ型の表や比較図を多用
  • Day3：制御構文 → フローチャートやアルゴリズムの視覚化を重視
  • Day4：配列 → 配列操作の図解やメモリイメージを含める

視覚的要素を強化する

テキストだけでなく、視覚的な要素を積極的に取り入れましょう：

  • 概念図や関係図
  • コードの実行結果のスクリーンショット
  • 表やチャートを使った比較
  • アルゴリズムのフローチャート

コード例を効果的に表示する

Javaのコード例を示す際は以下の点に注意してください：

  • シンタックスハイライトを使用する
  • 重要な部分を強調表示する
  • 簡潔で理解しやすいコード例を選ぶ
  • コメントを適切に入れて説明を補強する

時間配分を意識する

時間割に沿ったスライド構成を心掛けましょう：

  • 各セッションの重要度に応じたスライド数の配分
  • 休憩時間を明示したスライドの挿入
  • 演習時間と説明時間のバランス
  • 質疑応答や振り返りの時間の確保
