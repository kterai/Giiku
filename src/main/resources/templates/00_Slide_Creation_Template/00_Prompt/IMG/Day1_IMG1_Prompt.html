JDK・JRE・JVMの関係

JDK (Java開発キット): Javaアプリケーションを開発するためのツール群。コンパイラ(javac)やデバッガなどが含まれます。

JRE (Java実行環境): Javaアプリケーションを実行するために必要な環境。クラスライブラリやAPIが含まれます。

JVM (Java仮想マシン): バイトコードを実際に実行する仮想マシン。「Write Once, Run Anywhere」を実現します。
