If-Else Flow Chart Image Generation Specifications

Image generation prompt for educational diagram showing if-else statement control flow

Image Overview

Filename: lecture3_img01.png

Size: 1000px × 700px

Purpose: Educational diagram for Java programming course showing if-else statement flow control

Style: Clean, professional flowchart with clear directional flow

Design Specifications

Layout

  • Vertical flow from top to bottom
  • Clear directional arrows connecting all elements
  • Diamond shapes for decision points
  • Rectangle shapes for code blocks and operations
  • Rounded rectangle for start/end points

Color Scheme

  • White background (#FFFFFF)
  • Light blue for decision diamonds (#E3F2FD)
  • Light green for true paths (#C8E6C9)
  • Light red for false paths (#FFCDD2)
  • Light yellow for code blocks (#FFF9C4)
  • Dark gray for text and outlines (#333333)

Typography

  • Sans-serif font (Arial or Helvetica)
  • Title: 24pt bold
  • Main flowchart text: 16-18pt
  • Code examples: monospace font (Courier New), 14pt
  • All text in English

Content Elements

Main Title

"IF-ELSE Statement Control Flow"

Flowchart Elements (Top to Bottom)

  1. Start (rounded rectangle)
  2. Condition evaluation (diamond shape with "condition?")
  3. Two paths from condition:
    • TRUE path (to the left or right, labeled "true" or "yes") → "if block" rectangle
    • FALSE path (to the opposite side, labeled "false" or "no") → "else block" rectangle (if present)
  4. Both paths reunite after their respective blocks
  5. Continue to next statement (or End)

Variations to Include

  1. Simple if-else (one condition)
  2. if-else if-else (multiple conditions in sequence)
  3. Nested if-else (condition inside another condition)

Code Examples to Include

Simple If-Else

if (score >= 60) {
    System.out.println("Pass");
} else {
    System.out.println("Fail");
}

If-Else If-Else Chain

if (score >= 90) {
    grade = 'A';
} else if (score >= 80) {
    grade = 'B';
} else if (score >= 70) {
    grade = 'C';
} else {
    grade = 'F';
}

Nested If-Else

if (isLoggedIn) {
    if (isAdmin) {
        showAdminPanel();
    } else {
        showUserDashboard();
    }
} else {
    showLoginForm();
}

Special Instructions

  • Include directional arrows with clear flow direction
  • Label TRUE/FALSE paths clearly
  • Use different colors for TRUE/FALSE paths to enhance visual clarity
  • Include a legend explaining symbols and colors
  • Make text readable at a glance - keep it concise
  • Keep the overall design clean and minimalist

Image Use Context

This image will be used in Day 3 of the JavaSilver course, focusing on control flow statements. It will be the primary visual aid for explaining if-else statement execution flow to programming beginners.

The image should be clear enough to be understood without extensive explanation, but will be accompanied by verbal explanation from the instructor.
