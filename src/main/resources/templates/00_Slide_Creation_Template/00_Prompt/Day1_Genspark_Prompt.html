重要なコード例

1. よくある間違い例

// ❌ コンパイルエラーになる例 byte b = 100; b = b + 1; // int + int = int なので、byteに代入できない // ✅ 正しい書き方 byte b = 100; b += 1; // 複合代入演算子は自動キャスト b = (byte)(b + 1); // 明示的キャスト

2. String型の動作

// String literal pool の動作 String s1 = "Hello"; String s2 = "Hello"; String s3 = new String("Hello"); System.out.println(s1 == s2); // true (同じ参照) System.out.println(s1 == s3); // false (異なる参照) System.out.println(s1.equals(s3)); // true (同じ値)

3. 型変換の実例

// 暗黙の型変換（プロモーション） int i = 100; double d = i; // int → double は自動変換 // 明示的な型変換（キャスト） double d = 3.14; int i = (int)d; // double → int は明示的キャスト必要 // 文字と数値の変換 char c = 'A'; int ascii = c; // char → int は自動変換 System.out.println(ascii); // 65

JavaSilver試験対策ポイント

頻出ひっかけポイント

  • byte/short + int → int になることを忘れがち
  • char型は数値型として扱われる
  • String == と equals()の違い
  • final変数の再代入不可
  • 変数の初期化が必要な場面

覚えるべき数値

整数型の範囲

  • byte: -128 ～ 127
  • short: -32,768 ～ 32,767
  • int: -2,147,483,648 ～ 2,147,483,647
  • long: -9,223,372,036,854,775,808 ～ 9,223,372,036,854,775,807

文字のASCII値

  • 'A' = 65
  • 'a' = 97
  • '0' = 48
  • ' ' = 32

練習問題例（講義に含めるもの）

問題1（基本レベル）

次のコードでコンパイルエラーが発生するのはどれですか？

A. int x = 5; B. float y = 1.5f; C. double z = 2.0d; D. long l = 10000000000; E. boolean b = true;

正解: D

解説: 整数リテラルはデフォルトでint型ですが、この値はintの範囲を超えるため、Lサフィックスが必要です。

問題2（中級レベル）

次のコードの実行結果は何ですか？

String s1 = "Hello"; String s2 = "Hello"; String s3 = new String("Hello"); System.out.println(s1 == s2); System.out.println(s1 == s3); System.out.println(s1.equals(s3));

正解: true, false, true

解説: s1とs2は同じリテラルプールの参照、s3は新しいオブジェクト。equalsは値を比較。

問題3（難問レベル）

次のコードの実行結果は何ですか？

byte b = 100; b += 30; System.out.println(b); byte c = 100; c = c + 30; System.out.println(c);

正解: 130, コンパイルエラー

解説: 複合代入演算子は自動キャストしますが、通常の代入では明示的キャストが必要です。
